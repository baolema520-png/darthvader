Follow these steps in order. Check each one when you complete it.

0 of 5 steps completed

1. Install the app

Drag 520CAM→to Applications

The virtual camera only works when the app is installed in /Applications, not from the disk image or Downloads.

Open Applications

2. Open 520CAM

Open the app from Applications. If macOS warns about an identified developer, click Open.

Open 520CAM

3. Allow camera access

In the app, click Start. If macOS asks for permission, choose Allow. If you denied it before, enable it in Settings.

Open Camera privacy

4. Enable the virtual camera

Click Virtual Cam in the app. If macOS asks to approve the camera extension, go to Privacy & Security and allow it.

Open Privacy & Security

5. Select 520CAM in Zoom, Meet, or OBS

In your video app, choose 520CAM as the camera. If it does not appear, quit and reopen that app.

You can reopen this guide from the app using Setup help.
